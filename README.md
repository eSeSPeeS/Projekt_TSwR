# Projekt_TSwR

Model Predictive Control dla symulowanego samochodu

Cel projektu - stworzenie symulatora samochodu w oparciu o model dynamiczny opisany w publikacji naukowej arXiv:2003.04882.
Zaimplementowanie regulatora typu Model Predictive Control (MPC), który umożliwi śledzenie zadanych trajektorii ruchu pojazdu.


Projekt zostanie zrealizowany w języku Python z wykorzystaniem:
numpy – obliczenia numeryczne
matplotlib – wizualizacja wyników
do-mpc – implementacja regulatora MPC
scipy – narzędzia do optymalizacji i równań różniczkowych
